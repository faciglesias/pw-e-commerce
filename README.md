# pw-e-commerce

Proyecto inicial de e-commerce desarrollado con Next.js.

## Descripción del proyecto
Este proyecto tiene como objetivo crear una base para una tienda online, con un flujo de trabajo profesional usando GitHub, GitHub Actions y Vercel.

## Características
- Repositorio en GitHub
- Proyecto creado con Next.js
- Deploy automático en Vercel
- Pipeline CI/CD con GitHub Actions
- Base lista para seguir desarrollando el e-commerce

## Tech Stack
- Next.js
- React
- Node.js
- Vercel
- GitHub Actions
